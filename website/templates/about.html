About

lineinradio is a 24/7 online radio station that plays sets by djs, new albums by upcoming bands, and playlists submitted by listeners like you :)

how to get involved

if you would like to submit a set join the discord to become verified

if you would like to contribute code to the project, it's open source under the MIT Liscense on github
